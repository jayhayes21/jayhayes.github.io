My Projects:

Personal Projects

  • Made a web scraper using Python to fetch information from Strava to put into a csv file to track my weekly biking and running mileage. On top of that I track the number of times that I lift and add eliptical minutes onto my running mileage. Below you can see the Strava dashboard page and to the left it shows my weekly running mileage (6.0 mi).

  • Made a tool using Python to plot my weekly running miles, along with running mileage for high school and college friends, for either the past month or 2 months. The weekly running mile information will be pulled from an Excel sheet that I have on my Desktop.


  • Using Dark Sky API I created a Python script to track the weather on all of my runs. I record the running information in Strava where I then scrape it into a CSV file and attach an index number to each run. Using this index number I produce the weather for that run at an interval of my choosing. I created a shell script to be able to run this Python script along with the Python script to track my weekly cardio. Below is a sample output from the script.
